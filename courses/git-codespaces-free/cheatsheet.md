# GitHub Codespaces Cheatsheet

> "ไม่ต้องจำ command - ใช้ UI ได้หมด"

---

## เปิด Codespaces

### วิธี 1: Quick Edit (กด ".")
1. ไปที่ repo บน GitHub
2. กด `.` (จุด) บน keyboard
3. VS Code เปิดใน Browser ทันที!

### วิธี 2: Full Codespace
1. ไปที่ repo
2. กด `Code` (ปุ่มเขียว)
3. เลือกแท็บ `Codespaces`
4. กด `Create codespace on main`

---

## Commit + Push (ไม่ต้องพิมพ์ command)

```
1. แก้ไฟล์ตามปกติ

2. ไปแท็บ Source Control (ไอคอนกิ่งไม้ด้านซ้าย)

3. ดูไฟล์ที่เปลี่ยน → กด + เพื่อ Stage

4. พิมพ์ message ในช่อง "Message"

5. กด ✓ Commit

6. กด "Sync Changes" หรือ ↻
```

**แค่นี้! โค้ดขึ้น GitHub แล้ว**

## Keyboard Shortcuts

| Shortcut | ทำอะไร |
|----------|--------|
| `.` | เปิด VS Code web |
| `Ctrl+S` | Save file |
| `Ctrl+Shift+G` | เปิด Source Control |
| `Ctrl+`` ` | เปิด Terminal |

## Preview เว็บ

1. สร้างไฟล์ `.html`
2. คลิกขวา → "Open with Live Server"
3. หรือไปแท็บ `PORTS` → ดู URL

## ปิด Codespace (ประหยัด quota)

1. ไป github.com/codespaces
2. หา Codespace ที่เปิดอยู่
3. กด `...` → `Stop codespace`
