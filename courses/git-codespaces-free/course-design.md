### Part 2: Codespaces Magic (10:45-11:45)

**Goal:** VS Code บน Browser

| Time | Activity | Output |
|------|----------|--------|
| 10:45 | กด "." ใน repo = VS Code! | Mind blown |
| 10:55 | เปิด Codespaces | Full VS Code |
| 11:05 | Tour VS Code interface | รู้จัก UI |
| 11:20 | สร้างไฟล์แรก | First file |
| 11:35 | Commit + Push (ใน UI) | Code saved! |

**The Magic:**
```
วิธี 1: กด "." ใน repo = VS Code (web editor)
วิธี 2: กด "Create Codespace" = Full environment
```

**Commit ใน Codespaces:**
1. แก้ไฟล์
2. ไปแท็บ Source Control (ซ้าย)
3. พิมพ์ message
4. กด ✓ Commit
5. กด Sync

**ไม่ต้องจำ command!**
