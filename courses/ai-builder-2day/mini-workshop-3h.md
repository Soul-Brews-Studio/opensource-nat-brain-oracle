# AI Builder Mini Workshop - 3-4 Hours

> สำหรับเพื่อนถิงถิง (Online Session)
> Date: 2026-01-05, 11:00

---

## Overview

| Item | Detail |
|------|--------|
| **Duration** | 3-4 ชั่วโมง (รวม break) |
| **Format** | Online (Google Meet) |
| **Participants** | 2-3 คน + ถิงถิง |
| **Background** | ไม่เคยเขียนโค้ด |
| **Goal** | เห็นว่า AI ช่วยทำอะไรได้ + ลองทำเอง |

---

## Pre-Session Checklist

### ผู้เรียนเตรียม

- [ ] **ปัญหา/ไอเดีย 1 อย่าง** ที่อยากแก้ (ส่งมาก่อน)
- [ ] **บัญชี ChatGPT หรือ Claude** (free tier พอ)
- [ ] **ตัวอย่างไฟล์** (ถ้ามี) - PDF, รูป, Excel

### Instructor เตรียม

- [ ] Google Meet link
- [ ] Demo project (electrical calculator)
- [ ] Backup ideas สำหรับคนไม่มี

## Agenda (3.5 ชั่วโมง)

### Part 1: Opening (30 min) — 11:00-11:30

#### 11:00-11:10 | แนะนำตัว
- ทุกคนบอกชื่อ + ทำอะไร
- บอกปัญหา/ไอเดียที่เตรียมมา

#### 11:10-11:25 | Demo สด
แสดงให้ดู: **จากปัญหา → Prototype ใน 15 นาที**
- ใช้ตัวอย่าง electrical calculator (ของถิงถิง)
- อธิบายว่าเกิดอะไรขึ้น

#### 11:25-11:30 | Q&A
- ตอบคำถามแรก

---

### Part 2: First Contact (45 min) — 11:30-12:15

#### 11:30-11:40 | ตั้งค่า
- ทุกคนเปิด ChatGPT หรือ Claude
- Share screen ถ้าติด

#### 11:40-12:00 | ลองคุยกับ AI
ทุกคนทำพร้อมกัน:

```
1. พิมพ์ปัญหาของตัวเอง
2. ดู AI ตอบ
3. ถาม follow-up 2-3 รอบ
4. ลอง upload ไฟล์ (ถ้ามี)
```

#### 12:00-12:15 | Share
- แต่ละคนเล่าว่าได้อะไร
- ช่วยคนที่ติด

---

### Break (15 min) — 12:15-12:30

---

### Part 3: Deeper Dive (60 min) — 12:30-13:30

#### 12:30-12:45 | Concept: Context is Everything

**3 เรื่องสำคัญ:**
1. **ยิ่งให้ข้อมูลมาก ยิ่งได้คำตอบดี**
2. **Upload ไฟล์ได้** (PDF, รูป, Excel)
3. **ถามได้หลายรอบ** ไม่ต้องถูกตั้งแต่ครั้งแรก

**Demo:**
- ถามแบบไม่มี context → ได้คำตอบกว้าง
- ถามแบบ upload file → ได้คำตอบเฉพาะ

#### 12:45-13:20 | Build Something (35 min)

**เลือก 1 ใน 3:**

**Option A: Document Helper**
```
1. Upload เอกสารที่ทำงานอยู่
2. ถาม AI ให้ช่วยสรุป
3. ถาม AI ให้ช่วยเขียนใหม่
4. ถาม AI ให้ช่วยแปล
```

**Option B: Data Analyzer**
```
1. Upload Excel/CSV
2. ถาม AI ให้ช่วยวิเคราะห์
3. ขอ insights
4. ขอ visualization ideas
```

**Option C: Problem Solver** (ถ้ามีปัญหาเฉพาะ)
```
1. อธิบายปัญหาละเอียด
2. ให้ AI ถามคำถาม clarify
3. ได้ solution/plan
4. Iterate
```

#### 13:20-13:30 | Share Results
- แต่ละคนแชร์ว่าทำอะไร

---

### Part 4: Next Steps (30 min) — 13:30-14:00

#### 13:30-13:45 | Advanced Tips

**Tips ที่เป็นประโยชน์:**

1. **Custom Instructions** (ChatGPT)
   - บอก AI ว่าคุณเป็นใคร ทำอะไร
   - AI จะตอบเหมาะกับคุณมากขึ้น

2. **Project/Chat Organization**
   - แยก chat ตาม project
   - ตั้งชื่อให้หาง่าย

3. **Prompt Patterns ที่ใช้บ่อย**
   ```
   "ช่วย [action] ให้หน่อย โดย [constraint]"
   "ถ้าไม่เข้าใจตรงไหน ถามได้เลย"
   "ลองให้ 3 options"
   ```

#### 13:45-14:00 | Q&A + Closing

**แจก:**
- Resource list (links)
- Group chat access (Line/Discord)
- Follow-up session info

**ถาม:**
- อยากเรียนต่อไหม? (Full 2-day?)
- มีอะไรอยากทำเพิ่ม?
