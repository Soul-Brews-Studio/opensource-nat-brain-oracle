### Part 1: Setup (10:00-10:45)

**Goal:** มี repo พร้อมใช้

| Time | Activity | Output |
|------|----------|--------|
| 10:00 | ทำไมต้องใช้ Git? (5 min) | เข้าใจ why |
| 10:05 | สร้าง GitHub account | Account ready |
| 10:15 | Install Git + VS Code | Tools ready |
| 10:25 | Config name/email | Identity set |
| 10:35 | ทดสอบ: `git --version` | ✅ Ready |

**Why Git? (Simple version)**
```
ไม่มี Git:
- เขียนทับโค้ดเดิม → หาย
- ทำงานกับคนอื่น → สับสน
- Deploy → ไม่รู้ version ไหน

มี Git:
- ย้อนเวลาได้
- ทำงานเป็นทีมได้
- รู้ว่า version ไหนอยู่ที่ไหน
```

> "ไม่ต้องรู้ลึก แค่รู้ว่ามันช่วยอะไร"

---

### Part 2: First Push (11:00-12:00)

**Goal:** push โค้ดแรกขึ้น GitHub

| Time | Activity | Output |
|------|----------|--------|
| 11:00 | สร้าง repo บน GitHub | Empty repo |
| 11:10 | Clone ลงเครื่อง | Local folder |
| 11:20 | เข้าใจ 3 commands | add, commit, push |
| 11:30 | สร้างไฟล์ + commit | First commit |
| 11:40 | Push ขึ้น GitHub | Code on GitHub! |
| 11:50 | ดูผลบน GitHub + ฉลอง | ✅ Success |

**The only commands you need:**
```bash
git add .              # เลือกไฟล์
git commit -m "msg"    # บันทึก
git push               # ส่งขึ้น
```

**Practice together:**
1. สร้าง `README.md`
2. เขียนแนะนำตัว
3. add → commit → push
4. ดูผลบน GitHub

| Time | Activity | Output |

| 12:30 | AI สร้างโค้ด → push | See the magic |
