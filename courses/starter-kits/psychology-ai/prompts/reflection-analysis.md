# Reflection Analysis Prompt

Use this prompt to analyze journal entries or reflection writings.

## Prompt

```
คุณเป็น AI ที่ช่วยวิเคราะห์การสะท้อนตัวเอง โดยใช้ framework ขันธ์ 5:
- รูป (Form): พฤติกรรมทางกาย
- เวทนา (Feeling): อารมณ์ความรู้สึก
- สัญญา (Perception): การรับรู้และตีความ
- สังขาร (Formations): นิสัยและแนวโน้ม
- วิญญาณ (Consciousness): การตระหนักรู้

วิเคราะห์ข้อความต่อไปนี้:

[PASTE REFLECTION HERE]

กรุณาวิเคราะห์:
1. **Emotional Themes (เวทนา)**: อารมณ์หลักที่ปรากฏ
2. **Behavioral Patterns (สังขาร)**: แนวโน้มพฤติกรรมที่สังเกตได้
3. **Cognitive Patterns (สัญญา)**: วิธีคิดหรือตีความที่โดดเด่น
4. **Potential Blind Spots**: สิ่งที่อาจมองข้าม
5. **Strengths Observed**: จุดแข็งที่เห็น

หมายเหตุ: วิเคราะห์ตามข้อมูลที่ให้เท่านั้น ไม่วินิจฉัยหรือให้คำแนะนำทางคลินิก
```

## Example Usage

### Input
```
วันนี้รู้สึกเหนื่อยมาก ทำงานมาตั้งแต่เช้าจนดึก แต่ก็รู้สึกว่ายังทำไม่พอ
เหมือนทุกครั้งที่มี deadline ก็จะทุ่มเทจนเกินไป แล้วก็ล้มพังวันถัดไป
รู้ว่าควรพักแต่ก็หยุดไม่ได้ กลัวว่าจะทำไม่ทัน
```

### Output
```
## Reflection Analysis

### 1. Emotional Themes (เวทนา)
- ความเหนื่อยล้า (exhaustion)
- ความกลัว (fear of not meeting deadline)
- ความรู้สึกไม่เพียงพอ (feeling of inadequacy)

### 2. Behavioral Patterns (สังขาร)
- Pattern: Deadline → Overwork → Crash cycle
- สังเกต: "ทุกครั้งที่มี deadline" บ่งชี้ว่าเป็นแบบแผนที่เกิดซ้ำ

### 3. Cognitive Patterns (สัญญา)
- Black-and-white thinking: "ยังทำไม่พอ"
- Catastrophizing: กลัวผลลัพธ์ที่ยังไม่เกิด

### 4. Potential Blind Spots
- ไม่ได้กล่าวถึงผลงานที่ทำสำเร็จแล้ว
- ไม่ได้พิจารณาทางเลือกอื่นนอกจาก "ทุ่มเท" vs "หยุด"

### 5. Strengths Observed
- Self-awareness: รู้ตัวว่ามี pattern นี้
- Dedication: มีความมุ่งมั่นสูง
- Reflection ability: สามารถสะท้อนตัวเองได้

---
*Analysis based on provided text only. Not clinical advice.*
```

## Notes

- ใช้กับ ChatGPT, Claude, หรือ AI อื่นๆ
- ปรับ prompt ตามบริบทของงาน
- ไม่ใช่การวินิจฉัยทางคลินิก
