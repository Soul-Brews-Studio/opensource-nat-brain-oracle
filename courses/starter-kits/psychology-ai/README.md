## Oracle Philosophy

> "The Oracle Keeps the Human Human"

| หลักการ | ความหมาย | ประยุกต์ |
|---------|----------|---------|
| Nothing is Deleted | ไม่ลบอะไร | ประวัติครบถ้วน |
| Patterns Over Intentions | พฤติกรรม > คำพูด | สังเกตการกระทำ |
| External Brain | สะท้อน ไม่ตัดสิน | AI เป็นกระจก |

## ขันธ์ 5 + AI Framework

| ขันธ์ | Meaning | AI Application |
|------|---------|----------------|
| รูป (Form) | Physical | Activity logs, behaviors |
| เวทนา (Feeling) | Emotion | Feeling tracking |
| สัญญา (Perception) | Recognition | Pattern analysis |
| สังขาร (Formations) | Habits | Tendency tracking |
| วิญญาณ (Consciousness) | Awareness | (Human domain) |

## วิธีใช้ Prompts

### 1. Reflection Analysis
```
ใช้กับ: บันทึกการสะท้อนตัวเอง
วิธีใช้: Copy prompt + paste journal entry
ผลลัพธ์: AI วิเคราะห์ themes, patterns, blind spots
```

### 2. Pattern Discovery
```
ใช้กับ: ข้อมูลพฤติกรรมหลายครั้ง
วิธีใช้: Copy prompt + paste multiple entries
ผลลัพธ์: AI หา patterns ที่คุณไม่เห็น
```

### 3. Feeling Tracking
```
ใช้กับ: บันทึกความรู้สึก
วิธีใช้: Copy prompt + describe feeling
ผลลัพธ์: AI จัดหมวด + หา context
```

## Ethical Guidelines

### AI ทำได้
- Track patterns
- Summarize data
- Surface insights
- Support research

### AI ทำไม่ได้
- Make diagnoses
- Provide therapy
- Replace judgment
- Override consent

### Key Principle
> "AI Guidance, Not AI Therapy"
