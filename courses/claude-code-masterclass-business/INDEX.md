## Overview

- Business owners ที่อยากใช้ AI เพิ่ม productivity

- ใช้ Claude Code พื้นฐานได้ (chat, edit files, run commands)

## Modules

### Module 1: Sub Agents (2h)
> "ให้ AI จ้าง AI ทำงานแทน"

- Why delegate? Context efficiency
- Haiku vs Opus - when to use which
- Parallel task execution
- Real business case: bulk operations

### Module 2: Skills (1.5h)
> "สร้าง workflow ที่ใช้ซ้ำได้"

- SKILL.md anatomy
- Trigger phrases that work
- Business skill examples: report generator, email drafter
- Sharing skills across projects

### Module 3: Commands (1.5h)
> "Automate repetitive tasks"

- Slash commands basics
- .claude/commands/ structure
- Chaining commands
- Business workflows: /daily-report, /client-summary

### Module 4: MCP (Model Context Protocol) (2h)
> "เชื่อมต่อ AI กับระบบภายนอก"

- What is MCP?
- Available MCP servers
- Business integrations: CRM, databases, APIs
- Building custom MCP server (optional advanced)
