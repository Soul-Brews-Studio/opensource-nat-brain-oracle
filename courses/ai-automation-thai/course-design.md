## Session Details

### Session 1: The Automation Mindset (9:00-10:30)

| Time | Activity | Details |
|------|----------|---------|
| 9:00-9:30 | Opening Demo | สร้าง automation จาก 0 ใน 20 นาที (live) |
| 9:30-10:30 | 3 Questions Framework | INPUT → PROCESS → OUTPUT |

**3 คำถามสำคัญ:**

| Question | Ask | Example |
|----------|-----|---------|
| **INPUT** | อะไรเข้า? เมื่อไหร่? | New email, new row, scheduled |
| **PROCESS** | ทำอะไร? AI ช่วยตรงไหน? | Summarize, categorize, generate |
| **OUTPUT** | อะไรออก? ส่งไปไหน? | Slack message, new doc, database |

**Hands-on:** วิเคราะห์งานซ้ำๆ ของตัวเอง ด้วย 3 คำถามนี้

---

### Session 2: AI as Brain (10:45-12:00)

| Time | Activity | Details |
|------|----------|---------|
| 10:45-11:15 | AI Capabilities | READ / THINK / CREATE |
| 11:15-12:00 | Hands-on: AI Nodes | ต่อ AI เข้า workflow |

**AI ทำอะไรได้:**

| Capability | Input | Output | Example |
|------------|-------|--------|---------|
| **READ** | รูป, PDF, เสียง | Text, data | OCR, transcription |
| **THINK** | Text | Decision | Categorize, sentiment |
| **CREATE** | Prompt | Content | Summary, rewrite |

**Key Learning:** "AI ไม่ใช่ magic ต้องบอกให้ชัด"

---

### Session 3: Build Your First Automation (13:00-15:00)

| Time | Activity | Details |
|------|----------|---------|
| 13:00-13:30 | Choose Project | เลือก Level A/B/C |
| 13:30-15:00 | Build Session | สร้าง automation แรก |

**Project Levels:**

| Level | Complexity | Example Output |
|-------|------------|----------------|
| **A** | Simple | Trigger → AI → Output (email summary) |
| **B** | Multi-step | With decisions (content pipeline) |
| **C** | Complex | With integrations (full workflow) |

**Recommendation:** เริ่ม Level A ก่อน

---

### Session 4: Debug & Iterate (15:15-16:30)

| Time | Activity | Details |
|------|----------|---------|
| 15:15-15:45 | Common Problems | 4 ปัญหาที่เจอบ่อย |
| 15:45-16:30 | Fix Your Workflow | Test → Bug → Fix → Repeat |

**Common Problems:**

| Problem | Cause | Fix |
|---------|-------|-----|
| AI ไม่เข้าใจ | Prompt ไม่ชัด | Rewrite prompt |
| Wrong format | ไม่ specify | Add format instructions |
| Errors | No handling | Add conditions |
| Too slow | Too much data | Optimize flow |
