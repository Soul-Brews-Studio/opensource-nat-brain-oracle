## Oracle Visual Style Guidelines

All prompts emphasize:

- **Ink-wash base**: Monochrome fluid backgrounds with geometric precision
- **Cyan accents**: Semi-transparent highlighting, energy flows, state indicators
- **Gold accents**: Decision boundaries, completion markers, flow transitions
- **Geometric framework**: Technical precision underlying organic forms
- **Zen minimalism**: Careful use of space, purposeful composition
- **Thai typography**: Open source fonts only (Noto Sans Thai, Prompt, Kanit, Sarabun)
