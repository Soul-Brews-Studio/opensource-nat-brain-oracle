## File Naming Convention

All files follow this pattern:
```
antigravity-[NUMBER]-[slug].md
```

Example: `antigravity-301-nothing-is-deleted.md`

## File Structure

Each file contains:
- YAML frontmatter with metadata (type, target, status, created date, category, file_number)
- Markdown title (# Title)
- Thai text translations
- Complete visual prompt specifications
- Expected output format (PNG, 16:9, dimensions, filename)

## Usage

These prompts are intended for:
- **Image generation**: DALL-E, Midjourney, Stable Diffusion
- **Visual gallery**: Slideshow presentations of Nat's Agents philosophy
- **Documentation**: Accompanying written guides and tutorials
- **Reference**: Quick lookup of visual metaphors for concepts
