## üéØ What is Opal?

**Opal** is a new experimental feature from Google Labs that transforms prompts into **interactive mini apps** within Gemini.

### Key Concept
- Not just chatting with AI
- Converts prompts ‚Üí Interactive Mini Apps
- Reusable AI tools with multi-step workflows
- Works as "Experimental Gem" in Gemini Web App

---

## ‚ú® Key Features

### 1. In-Chat App Creation
- Create AI tools directly from Gems Manager
- Build custom AI assistants without coding

### 2. Visual Editor
- **New perspective**: View prompts as "List of steps" instead of long text
- See workflow/flow clearly
- Edit steps easily
- Better than traditional long prompt engineering

### 3. Advanced Mode
- Deep customization available at: **opal.google**
- For detailed tweaking

### 4. Availability
- Currently: Desktop only
- Status: Experimental (Google Labs)
