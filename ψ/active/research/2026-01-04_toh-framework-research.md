## Executive Summary

Toh Framework is a **free, open-source SaaS Builder AI Agent Framework** (v1.7.0) that enables non-programmers to build production-ready web applications using natural language. Created by Wasin Treesinthuros, it positions itself as a DIY tool for rapid MVP development.

**Key Finding**: Toh Framework is **complementary, not competitive** to our courses. It's a tool (like Claude Code), while we teach skills that work with any tool.

## Recommendations

### Don't Compete On
- Price (they're free, we charge for education)
- Framework features (different product types)
- "Build apps fast" alone (they have this too)

### Compete On
1. **Skill Transfer** - Learn once, use forever with any tool
2. **Independence** - Not locked to one framework
3. **Understanding** - Know WHY, not just HOW
4. **Flexibility** - Build solutions Toh doesn't support
5. **Live Support** - Real instructor, real-time help

### Consider
- **Partnership potential**: Toh Framework could be ONE of the tools we teach
- **Market education**: They're teaching market that AI building is possible
- **Different segments**:
  - Toh attracts: "I want to build this one thing fast"
  - We attract: "I want to learn how to build anything"
