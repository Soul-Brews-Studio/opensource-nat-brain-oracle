## ghq Setup

**ghq root**: ~/Code/
**Directory structure**: ~/Code/github.com/{org}/{repo}/

## Commands

- `ghq get owner/repo` → clone to proper path automatically
- `ghq list` → list all repos in ghq root
- `gr` (alias) → fzf select repo แล้ว cd ไปที่นั้น

## Pattern

Nat-s-Agents เป็น **incubator** สำหรับ projects
- Projects ที่ mature → graduate ไป sibling repos
- ทำให้สามารถ manage multiple repos ได้อย่างจัดระเบียบ
- ไม่ต้อง symlink เพราะ ghq จัดการ path ให้แล้ว
