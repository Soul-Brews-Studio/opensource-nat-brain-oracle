# Prefer Meetup/Event, Not Call

## Context

In the Dr. Do draft message, I wrote:
> "ถ้าอยากลองคุย call กันสักรอบก็ได้"

This is wrong. I don't prefer calls.

## The Preference

| Don't Prefer | Prefer |
|--------------|--------|
| 1:1 call | Meetup |
| Video call | Event |
| Phone call | In-person gathering |
| Scheduled call | Community session |

## Why

- Calls are **draining** (1:1 energy extraction)
- Meetups/events are **multiplying** (community energy)
- Calls feel **transactional**
- Events feel **relational**

## Update Dr. Do Message

Change:
> ~~"ถ้าอยากลองคุย call กันสักรอบก็ได้"~~

To:
> "ถ้ามี event หรือ meetup ที่น่าสนใจ จะบอกให้นะครับ"
> "หรือถ้าอยากเจอกัน ก็มา community live session ที่กำลังจะจัดได้"

## Pattern

This connects to earlier insight:
- "1:1 calls drain, live sessions multiply"
- Community > Individual
- Friendship > Transaction
