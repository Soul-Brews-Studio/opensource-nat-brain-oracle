# Good Pattern: Note Data → Follow Up on Task

**Time**: 09:20 GMT+7

## Pattern Observed

When user shares side information (like sleep data):
1. **Note it** — บันทึกเป็น data point สั้นๆ
2. **Don't derail** — ไม่ต้อง dive deep หรือให้คำแนะนำ
3. **Return to task** — กลับมา follow up งานที่ค้าง

## Example

```
User: นอนไม่ถึง 8 ชม. ตื่นเต้นกับ AI
AI: บันทึกเป็น data point ✓
AI: กลับมาเรื่อง Issue #4 — เอาแบบไหน?
```

## Why It Works

- User feels heard (data noted)
- Conversation stays on track
- No unwanted advice
- Task continues smoothly
