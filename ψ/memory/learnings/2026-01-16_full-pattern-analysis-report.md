## Part 5: Calculations Used

### Work Hours Formula

```python
work = window - sleep

where:
  window = last_commit_time - first_commit_time
  sleep = biggest_gap if biggest_gap > 4h else 0
```

### DuckDB Queries Used

```sql
-- Time Classification
SELECT
    CASE
        WHEN EXTRACT(HOUR FROM timestamp) BETWEEN 6 AND 11 THEN 'Morning'
        WHEN EXTRACT(HOUR FROM timestamp) BETWEEN 12 AND 17 THEN 'Afternoon'
        WHEN EXTRACT(HOUR FROM timestamp) BETWEEN 18 AND 23 THEN 'Evening'
        ELSE 'Night'
    END as time_period,
    COUNT(*) as commits
FROM commits
GROUP BY 1

-- Commit Type Distribution
SELECT
    CASE
        WHEN message LIKE 'rrr:%' THEN 'Retrospective'
        WHEN message LIKE 'feat%' THEN 'Feature'
        WHEN message LIKE 'fix%' THEN 'Fix'
        ELSE 'Other'
    END as commit_type,
    COUNT(*) as count
FROM commits
GROUP BY 1

-- Weekly Trend
SELECT
    DATE_TRUNC('week', timestamp)::DATE as week,
    COUNT(*) as commits
FROM commits
GROUP BY 1
ORDER BY 1
```
