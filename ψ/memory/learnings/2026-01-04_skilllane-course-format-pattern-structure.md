# ## SkillLane Course Format Pattern

## SkillLane Course Format Pattern

### Structure (from CodingThailand reference)
1. Course title + slug
2. Price (sale/regular) 
3. Quick stats: videos, duration, lifetime access, downloads, articles
4. "สิ่งที่ได้รับ" - 3-4 bullet points
5. Course description paragraph
6. "ประโยชน์ที่ผู้เรียนจะได้รับ" - detailed bullet list
7. "ผู้เรียนควรมีพื้นฐานอะไรมาก่อน"
8. "เนื้อหา" - video list with durations
9. "ผู้สอน" - instructor profile with ratings
10. "ห้องสนทนา", "วิธีการชำระเงิน", "คอร์สแนะนำ"

### Our Differentiators vs Competitors
- **Philosophy**: "สอนตกปลา ไม่ใช่ให้ปลา" (Teach fishing, not give fish)
- **Focus**: No-code AI building (vs coding tutorials)
- **Target**: Business owners (vs developers)
- **Outcome**: Skills that transfer to any tool (vs tool-specific templates)
