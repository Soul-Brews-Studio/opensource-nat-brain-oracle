# React Testing Library best practices: test behavior not implementation. Use getB

React Testing Library best practices: test behavior not implementation. Use getByRole over getByTestId. Prefer userEvent over fireEvent for realistic interactions. Avoid testing implementation details like state or lifecycle methods.
