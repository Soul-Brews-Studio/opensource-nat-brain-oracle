# Push notifications: Use FCM (Firebase Cloud Messaging) or Web Push API. Request 

Push notifications: Use FCM (Firebase Cloud Messaging) or Web Push API. Request permission at meaningful moments, not on page load. Store push subscriptions server-side. Service worker handles background notifications. Implement notification preferences per user. Use silent pushes for data sync. Test notification delivery across platforms.
