# **Gemini Proxy Debug Topics**: Use multiple MQTT topics for granular state track

**Gemini Proxy Debug Topics**: Use multiple MQTT topics for granular state tracking:
- `claude/browser/state/icon` - red/green changes
- `claude/browser/state/inject` - content script injection success/fail
- `claude/browser/state/connection` - websocket status
- `claude/browser/log/debug` - verbose logging when needed

Allows selective subscription for debugging vs production use.
