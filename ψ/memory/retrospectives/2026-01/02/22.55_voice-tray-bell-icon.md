## Technical Note: PNG32 for RGBA

**Problem**: Normal ImageMagick conversion doesn't always produce RGBA

```bash
# This may NOT produce RGBA:
magick source.png -resize 32x32 -alpha set output.png

# This ALWAYS produces RGBA:
magick source.png -resize 32x32 -alpha on PNG32:output.png
```

**Verification**:
```bash
file output.png
# Should say: "8-bit/color RGBA"
```
