## Patterns Observed

### 1. Iterative Refinement Works
- ทำทีละจุด ไม่ต้องเสร็จทีเดียว
- ปรับ → preview → ปรับอีก → จนพอใจ
- User นำ direction, AI เสนอ options

### 2. Format Matters
- Markdown heading levels affect readability
- `<br>` for packing lines together
- Blockquote `>` for visual grouping
- Arrow `→` format ดี แต่ descriptive prose ก็ได้

### 3. Thai + English Mix
- Title: Thai + English works well
- Tagline: Thai feels more personal
- Technical terms: Keep English (Agentic AI, Claude Code)

### 4. Content vs Structure Separation
- `raw/` = untouched source
- `reference/` = processed for use
- `projects/` = human work in progress
- `ψ/outbox/` = AI-ready output (ไม่ใช่ human collab)
