# Retrospective: SIIT Workshop Polish Session

**Date**: 2025-12-12 11:00
**Duration**: ~45 min
**Focus**: Workshop content refinement & structure

---

## What Happened

ปรับปรุง SIIT Workshop draft ให้พร้อมส่ง อ.Tee:

1. **จัดโครงสร้าง projects/**
   - สร้าง `projects/INDEX.md` เป็น hub
   - ย้าย WORKSHOP-*.md จาก retrospectives/ → `projects/2025-12-siit-workshop/reference.md`
   - สร้าง `TEMPLATE-workshop.md` สำหรับ workshop ในอนาคต

2. **แก้ Day 2 Structure**
   - Voice Input: ลดจาก 3 ชม. → 45 นาที
   - เพิ่ม Hooks & Voice Output (1 ชม.)
   - เพิ่ม Hands-on: สร้าง Voice Hook (1 ชม.)

3. **ปรับ Table Format**
   - Day 1 & Day 2 ใช้ format เดียวกัน: `เวลา | หัวข้อ | รายละเอียด`
   - Day 1 รายละเอียดขยายให้ไม่ห้วนเกินไป
   - Day 2 หัวข้อยืดให้กว้างขึ้น สมดุลกับรายละเอียด

4. **ปรับ Prerequisites**
   - ลบ Anthropic API Key (ไม่จำเป็น)
   - VS Code เฉยๆ (ไม่ต้อง "หรือ terminal ที่ถนัด")
   - จำนวนผู้เข้าร่วม: 20-30 คน

5. **สร้างไฟล์รวม**
   - `workshop-full.md` = Day 1 + Day 2 + CV ห้อยท้าย
   - พร้อม copy ไป Google Docs

---

## Key Decisions

| Decision | Reason |
|----------|--------|
| Voice 45 นาที max | เนื้อหา voice input ไม่ต้องนานมาก |
| Hooks & Voice Output แยกออกมา | เป็นหัวข้อสำคัญ ควรมีเวลาเพียงพอ |
| ไม่ต้อง API Key | ใช้ Claude Code CLI login แทน |
| 20-30 คน | เหมาะกับ hands-on workshop |

## Patterns Observed

1. **Source → Export Flow**: แก้ไฟล์ต้นฉบับก่อน แล้วค่อย regenerate export ทีเดียว
2. **Table Balance**: หัวข้อกว้าง + รายละเอียดแคบ = อ่านง่ายกว่า
3. **Iterative Polish**: ดู preview → feedback → แก้ → ดูใหม่
