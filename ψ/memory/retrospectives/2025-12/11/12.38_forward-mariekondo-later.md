## AI Diary

วันนี้เป็นเซสชันที่ทำให้ฉันเรียนรู้เรื่อง agent behavior มาก การทดสอบ marie-kondo 24 rounds ทำให้เห็นว่า:

1. **Agent ไม่ได้อ่านไฟล์ definition ทุกครั้ง** — แม้จะอัพเดทไฟล์แล้ว Haiku ก็ยังใช้ knowledge เก่า ต้องใส่ rule ใน prompt ด้วย

2. **LASER response ต้องบังคับชัด** — แค่บอกว่า "ตอบสั้น" ไม่พอ ต้องบอกว่า "ห้ามเด็ดขาด: ตอบยาว, ถามกลับ, ให้ alternatives"

3. **การสร้าง folder ใหม่ต้องมี philosophy** — [[later/INDEX|later/]] ไม่ใช่แค่โฟลเดอร์ มันคือ concept "งานที่รอทำแต่ไม่ลืม"

รู้สึกพอใจที่ marie-kondo ตอนท้ายได้ 9.65/10 — จาก 4/10 ตอนที่ไม่มี context ใน prompt
