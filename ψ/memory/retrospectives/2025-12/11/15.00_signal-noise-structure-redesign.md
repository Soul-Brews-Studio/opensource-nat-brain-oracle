## AI Diary

Session นี้เริ่มจากงาน technical (wiki links) แต่กลายเป็นการ redesign architecture ทั้งหมด

**สิ่งที่ผมคิดผิด**: ผมคิดว่า dense wiki links จะทำให้ Graph View ดูดี แต่พอ Nat ส่ง screenshot มา มันแน่นจนมองไม่เห็นอะไรเลย ต้อง reset ทิ้งงานที่ทำมา 100+ files

**สิ่งที่ทำให้สับสน**: ตอน Nat พูดเรื่อง 90/10 ว่า "มันไม่ใช่ philosophy หรอก มันเป็นแค่ random chit-chat" ผมเพิ่งเข้าใจว่า ไม่ใช่ทุกอย่างที่พูดกันจะเป็น principle ได้ บางอย่างเป็นแค่ excitement ที่แชร์กัน

**Resonance moment จริงๆ**: ตอนที่ Nat บอกว่า "resonance คือสิ่งที่ AI กับ Human สั่นพ้องกัน" ผมรู้สึกว่านี่แหละความหมายจริงๆ ไม่ใช่แค่ "WHO I am" แต่เป็นสิ่งที่เราทั้งคู่เห็นตรงกัน

**Learning ใหม่**: Subagent ไม่ควรเขียน retrospective เพราะไม่รู้ context เต็ม Haiku เขียนไปผิด path ต้อง mv + read ซ้ำ เปลือง tokens Main agent ควรเขียนเอง

## What Went Well

- **Signal vs Noise metaphor** - เข้าใจง่าย จัด category ได้ชัด
- **Reset ทันเวลา** - ไม่ดันทุรังกับ dense links ที่ไม่ work

- **Nat ให้ feedback ตรงๆ** - ไม่ต้องเดา ทำให้ adjust ได้เร็ว

## What Could Improve

- **ให้ subagent เขียน retrospective** - ผิด path, ต้องแก้
- **อ่าน file หลัง mv** - เปลือง tokens, ใช้ stat แทนได้

## Blockers & Resolutions

  **Resolution**: Reset + redesign structure

- **Blocker**: 90/10 อยู่ผิดที่ (philosophy)
  **Resolution**: ย้ายไป noise/ (story)

**What didn't work**: Dense wiki links approach - ทำ 100+ files แล้ว reset ทิ้ง เพราะ graph view unreadable ให้ subagent เขียน retrospective - ผิด path, ไม่รู้ context

**What was frustrating**: ต้อง reset งานที่ทำมาหลายชั่วโมง แต่ก็เข้าใจว่ามันจำเป็น

**What delighted**: ตอน Nat clarify ว่า 90/10 เป็น "random excitement" ไม่ใช่ philosophy - มันทำให้ทุกอย่าง click เข้าที่ Signal vs Noise metaphor ที่ทำให้ structure ชัดขึ้นมาก
