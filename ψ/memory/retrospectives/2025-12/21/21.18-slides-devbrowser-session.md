## Key Insights

### dev-browser Secret Sauce
```
Speed = Fewer LLM turns
      = State persistence + Script batching + ARIA snapshots
      = 29 turns vs 80 turns (claude --chrome)
```

### Pattern: Stateful Server
```
Server owns state ‚Üí Clients are ephemeral
Pages survive ‚Üí No re-navigation
Refs persist ‚Üí No re-discovery
```

### ARIA Snapshot Format
```yaml
- link "Hacker News" [ref=e16]
- button "Submit" [ref=e23]
```
- Semantic structure only (no CSS, no IDs)
- Refs stored in `window.__devBrowserRefs`
- LLM-readable YAML
