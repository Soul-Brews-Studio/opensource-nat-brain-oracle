## Decisions Made

**Decision**: Create two separate decks instead of one combined

**Reasoning**:

## Lessons Learned

### 1. Slide Decks as Complete Prompts

**Discovery**: Slide decks can be prompts for AI to generate presentations

**Pattern**:
- Don't create slides manually
- Create prompts that AI can execute
- Include ALL content (no placeholders)
- Specify exact design requirements

**Benefit**: Faster creation, consistent quality, reusable

### 2. Modular Content Design

**Observation**: Two separate decks > one combined deck

**Reasons**:
- Independent usage
- Different skill levels
- Easier to customize
- Better focus

**Application**: When creating teaching materials, think modules not monoliths

### 3. Image Placeholders Enable Flexibility

**Pattern**: Mark image locations but don't generate immediately

**Format**: `[IMAGE PLACEHOLDER: description of needed visual]`

**Benefits**:
- User can choose style
- Can generate later
- Clear specifications
- No wasted work if style changes

### 4. Teaching Breakthrough Through Stories

**Insight**: Inter-Claude slides use actual breakthrough narrative

**Structure**:
- Real problem we faced
- Iterations and failures
- Discovery moment
- Proof it works
- How to replicate

**Why powerful**: Authentic story > theoretical explanation

### 5. Foundation Before Advanced

**Discovery**: tmux basics deck enables inter-Claude deck

**Pattern**:
- Deck 1: Foundation (tmux basics)
- Deck 2: Application (inter-Claude communication)

**Benefit**: Learners can progress naturally, or skip ahead if ready
