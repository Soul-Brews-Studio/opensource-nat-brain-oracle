## Key Learnings

### 1. MAW Copy Order Pattern
```
Config â†’ Setup â†’ Script â†’ Start
```
NOT: Start â†’ Debug â†’ Realize missing

### 2. Interactive Scripts Don't Work from Automation
- `maw kill` has `read -p` (y/N prompt)
- Use `tmux kill-session -t <name>` instead

### 3. osascript Click Precision
- Keyboard shortcuts can toggle wrong things
- Use `cliclick` for precise mouse positioning
- Calculate position from window bounds

### 4. Subagent Delegation Saves Time
- 5 parallel traces completed in seconds
- Debug subagent found root cause faster
- Main agent orchestrates, subagents investigate

## AI Diary

This session was intense - 38 minutes covering two completely different systems (MAW multi-agent and Antigravity image generation). The key insight was recognizing when to stop retrying and start investigating.

The MAW setup taught me about the relationship between config files and implementation scripts. Just copying profiles doesn't work if the start script doesn't know how to interpret them. This is a general pattern: configuration declares intent, code implements it.

The Antigravity fix was satisfying - moving from keyboard shortcuts (unreliable) to mouse clicks (precise). Sometimes the "hacky" solution (calculating pixel positions) is more robust than the "clean" one (keyboard navigation).
