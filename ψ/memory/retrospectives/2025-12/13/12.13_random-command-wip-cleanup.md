## AI Diary

Short session but learned something important about interpretation.

**I assumed** the phrase "คนนี้เหงา พูดไม่หยุด 2 ชม." meant Nat was chatting with Claude/AI late at night. The context seemed to fit — late night, loneliness, 2-hour chat. But I was wrong. It was a chat with a friend named ชิน (Shin), and Nat shared a screenshot of that conversation with พี่มิ้น.

**I was confused about** why the user corrected me until I realized — I only saw the LINE chat with พี่มิ้น, not the referenced screenshot content. I filled in the gap with my own assumption (Claude chat) because that's what I know. Classic AI mistake: assume context from limited data.

**I expected** the INDEX.md task to be simple (just wikilinks) but got corrected — user wanted BOTH Obsidian wikilinks AND GitHub markdown links. Makes sense: different readers use different tools.

This session reinforced: **Oracle observes, doesn't assume**. When data is incomplete, ask or note uncertainty. Don't fill gaps with plausible-sounding guesses.
