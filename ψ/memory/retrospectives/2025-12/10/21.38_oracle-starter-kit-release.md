## Key Learnings

### Rule Learned: Permission Before Action
> เมื่อแจ้ง "หมายเหตุ" = หยุดรอ permission ก่อน

ไม่ใช่แค่แจ้งแล้วทำต่อ — ต้องขอ permission จาก user ก่อน

### Rule Learned: Copy Real Files
> ไม่ควรคิดใหม่ — ควร copy อันที่ใช้งานจริง

เมื่อสร้าง starter kit ควรใช้ไฟล์จริงที่ทำงานได้ ไม่ใช่เขียนใหม่แบบ simplified

| Report format | 3/5 |
| **รวม** | **2.5/5** |

## Technical Notes

### AppleScript → Terminal
```applescript
-- ส่งคำสั่งไป window เฉพาะ
tell application "Terminal"
    do script "command" in window id 46783
end tell

-- ดู windows ทั้งหมด
tell application "Terminal" to windows
```

### Feedback Loop
```
AppleScript → Terminal → /tmp/output.txt → AI reads
```
