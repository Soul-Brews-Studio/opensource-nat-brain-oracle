# Hoppy Weiss — Styrian Wolf Dry Hop

**Date**: 2025-12-13
**Base**: Hoppy Weiss with T58 → ออกรสเปรี้ยว
**Dry Hop**: Styrian Wolf 10g/L
**Purpose**: Sponsor beer for Claude Code Meetup Bangkok!

## Tasting Notes

| Aspect | Note |
|--------|------|
| Overall | แปลกๆ แปร่งๆ เหมือนผงชูรส |
| Body | บาง |
| Drinkability | กินได้ง่ายกว่าตอนไม่ใส่ hop |
| After Taste | ยาว (น้ำมัน hop เยอะ) |
| Aroma | grassy ใหม่ๆ, เกือบ melon แต่ไม่ถึง |
| Ester | กลิ่นกล้วยจางๆ โดน hop dominate |
| Phenol | clove กานพู คลอๆมา |
| Hop | Old World style, น้ำมันเยอะ, ไม่ harsh |
| Hop Burn | เผ็ดเผ็ดที่หนวดเล็กน้อย |

## Verdict

- ดื่มยาวๆ = แปลกไป ตั้งคำถามไป แต่ดื่มได้เรื่อยๆ
- Pairing กับ pizza = ชิลๆ
- Aging ปีหน้า = จะเป็นอีกแบบ

**TL;DR**: Good beer! Not bad, but it'll be even better next year.

## Lesson Learned

> "beer จะทำให้สัมปชัญญะลดลง การสื่อสารจะเป็นแบบ emotional มากขึ้น
> ซึ่งมันผิดได้ อาจจะทำให้รู้สึกอายหรือเขินในภายหลัง
> แต่ว่า... นั่นคือความจริงที่เกิด เราก็เรียนรู้จากสิ่งพวกนี้กันไป"
