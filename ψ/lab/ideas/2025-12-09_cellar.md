## Concept

A lightweight desktop app that:
- Lives in system tray (menubar on Mac)
- Local SQLite database
- Stores command/task/session data
- Fast, smooth, always available

Name: **Cellar** — เก็บของดีๆ ไว้ใต้ดิน, บ่มได้ (history), brewing-inspired

## Tech Stack

- **Framework**: Flutter (cross-platform)
- **Database**: SQLite (sqflite / drift)
- **System Tray**: system_tray / tray_manager package
- **Platform**: macOS first, then Windows/Linux

---

## Why Flutter?

- Cross-platform from day one
- Good SQLite support
- Fast development
- Familiar (vs Tauri/Rust learning curve)
- Hot reload for iteration
