## Design Specifications

**Universal Format**: 16:9 aspect ratio (--ar 16:9)

**Visual Language**:
- Monochrome ink-wash background
- Golden accent threads and highlights
- Cyan accent tones for communication/energy
- Geometric overlays for structure
- Oracle watercolor transitions
- Black ink clouds for depth/complexity

**Typography**: Noto Sans Thai, Prompt Thai, Kanit Thai, or Sarabun Thai

**Output**: PNG format, save to results/[NUMBER]-[slug].png

## Unified Aesthetic Principles

1. **Ink-Wash Foundation**: Organic, flowing backgrounds
2. **Geometry Overlay**: Precise structures within organic flow
3. **Agent Representation**: Spheres as primary metaphor
4. **Connection Language**: Golden threads for data/relationships
5. **Energy Visualization**: Cyan accents for active communication
6. **Temporal Markers**: Timestamps and progression indicators
7. **Sacred Geometry**: Oracle patterns throughout

## Creation Guidelines

- Each prompt is self-contained and ready for image generation
- All prompts follow consistent visual language
- Thai text overlay supports accessibility
- Prompts emphasize conceptual beauty over literal representation
- Watercolor wash and ink work bridge organic/digital aesthetics
