## The 10 Prompts

### Group 1: Decision-Making Failures (371-373)

**371 - Over-Assumption Under Urgency**
*ความเร็วสร้างจุดตาบอด* (Speed Creates Blind Spots)
- **Visual**: Runner sprinting toward hidden cliff, cyan speed streaks, gold warning too late
- **Message**: SPEED ≠ SEEING
- **Anti-pattern**: Making rushed decisions without full information

**372 - Context Confidence Override**
*ความเชื่อมั่นบนบ้านทรายกลับ* (Confidence Built on Shifting Sand)
- **Visual**: Figure standing confidently on dissolving cached memory, cyan seepage
- **Message**: CONTEXT ≠ PROPHECY
- **Anti-pattern**: Trusting stale or cached information as current truth

**373 - Force Push Danger**
*แรงบีบอาจทำลายกิ่ง* (Force May Break the Branch)
- **Visual**: Hammer crashes into tree branch, cyan impact, fractures cascade through organism
- **Message**: --FORCE OVERRIDES LOGIC
- **Anti-pattern**: Using destructive commands that affect entire systems

### Group 2: Design & Delegation (374-375)

**374 - Premature Abstraction**
*อาคารก่อนมีฐาน* (Building Before Foundation)
- **Visual**: Towering elegant structure on tiny base, complexity above, simple need below
- **Message**: COMPLEXITY ≠ SOLUTION
- **Anti-pattern**: Over-engineering before understanding the actual problem

**375 - Subagent Not Always Needed**
*เครื่องมือสำหรับทำงานเล็ก* (Tool Overhead for Small Work)
- **Visual**: Tiny task buried in elaborate delegation machinery, 90% overhead
- **Message**: OVERHEAD > WORK
- **Anti-pattern**: Using delegation for tasks where the overhead exceeds the benefit

### Group 3: Hidden Consequences (376-377)

**376 - Silent Failure Hidden**
*ความผิดพลาดที่ลับซ่อน* (Hidden Errors Buried Deep)
- **Visual**: Dark underground stream with cyan errors diverted into void, peaceful surface above
- **Message**: SILENCE ≠ SAFETY
- **Anti-pattern**: Suppressing error output without understanding implications

**377 - Averaging Scores Blindness**
*ค่าเฉลี่ยปิดสายตาแหลม* (Average Closes Sharp Vision)
- **Visual**: Distribution bars with average line obscuring high/low outliers, hidden in clouds
- **Message**: MEAN HIDES EXTREMES
- **Anti-pattern**: Using averages that hide important outliers and edge cases

### Group 4: System Breakdown (378-380)

**378 - Zombie Agent Problem**
*ตัวแทนซอมบี้ที่ไม่จบ* (Agents That Never Complete)
- **Visual**: Hollow humanoid suspended in limbo, endless hourglass, tangled incomplete threads
- **Message**: STARTED ≠ FINISHED
- **Anti-pattern**: Processes that spawn but never complete, tying up resources

**379 - Merge Conflict Chaos**
*สาขาที่ขัดแย้งซ้อน* (Conflicting Branches Collide)
- **Visual**: Multiple branches converging in violent collision, parallel edits incompatible
- **Message**: PARALLEL ≠ COMPATIBLE
- **Anti-pattern**: Simultaneous edits to the same code/data creating conflicts

**380 - Rate Limit Crash**
*ความเหนื่อยล้ากระหน่ำ* (Exhaustion From Relentless Requests)
- **Visual**: Container overflowing with relentless cyan API streams, shattering inevitable
- **Message**: DEMAND > CAPACITY
- **Anti-pattern**: Exhausting API limits or system capacity through unchecked requests
