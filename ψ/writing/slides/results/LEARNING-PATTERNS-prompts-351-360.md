```markdown
- Final point at center glowing gold
- Decision crystallized
- Surrounded by supporting elements in lighter tones
- Thai annotation: "การตัดสินใจสุดท้าย" (final decision)
```

```markdown
# Fast Context Pattern
## ข้ามการค้นหา เมื่อเป้าหมายชัดเจน

### Thai Text
เป้าหมายชัดเจน → ข้ามการค้นหา → ตรงไปสู่การดำเนิน → เสร็จเร็ว
(Pao-hmai chat-jen → khaam kaan khon-haa → trong pai → sej reo)

**English**: Clear goal → Skip search → Direct execution → Done fast

### Prompt
Time-comparison ink-wash illustration showing two paths:

**SLOW PATH (WINDING)** - shown as faded background:
- Long zigzag path from start to finish
- Multiple search waypoints, detours, loops
- Takes full width of frame, many turns
- Labeled with clock showing "longer time"
- Muted gray colors suggesting inefficiency

**FAST PATH (DIRECT)** - shown in bold foreground:
- Straight cyan line from start (point A) to goal (point B)
- No detours, no searching
- Illustrated by an arrow moving at high speed
- Gold finish line at destination glowing bright
- Thai text "เร็ว" (fast) emphasized along the arrow
- Timer showing "shorter time" with gold checkmark

**KEY VISUAL ELEMENT**:
At the START point, a light bulb or clarity moment shown in gold - representing "clear goal". This triggers the direct cyan path. No searching needed when destination is already known.

Surrounding context suggests different scenarios:
- "เมื่อทราบเป้าหมาย" (when goal is known) - FAST PATH
- "เมื่อค้นหาเป้าหมาย" (when searching for goal) - SLOW PATH

The contrast emphasizes: discovering goal = searching; having goal = executing.

**16:9 composition**: Two overlapping paths with slow/fast contrast, clarity moment at start triggering the direct cyan line, gold finish at destination.
```

## Generation Notes

All prompts follow the style guide:
- **Base**: Monochrome ink-wash
- **Energy/Flow**: Cyan (#00FFFF or similar)
- **Completion/Decision**: Gold (#FFD700 or similar)
- **Aspect**: 16:9 PNG
- **Typography**: Noto Sans Thai, Prompt, Kanit, Sarabun only
- **Text**: Bilingual Thai + English translations with romanized phonetics

**Thai Typography Guide**:
- Main headings: Kanit (serif, traditional)
- Annotations: Prompt (modern, clean)
- Body/explanations: Sarabun (readable, friendly)
- English: Noto Sans (consistent international look)

**Visual Principles Applied**:
1. Each prompt shows a learning pattern or delegation principle
2. Ink-wash base prevents over-polished corporate feel
3. Cyan lines show flow, movement, connection
4. Gold marks completion, decision points, truth revealed
5. Thai text integrated as visual element, not just translation
6. Fractal/geometric patterns suggest self-similarity and emergence
7. Clear before/after or multiple perspectives where applicable
8. Mandala-like compositions for wholeness/system views

**Execution Tips for Image Generator**:
- Specify "ink-wash background" at start of each prompt
- Include "16:9 aspect ratio, PNG format"
- Emphasize "cyan (#00FFFF) for energy, gold (#FFD700) for completion"
- Request "Thai text rendered as calligraphic elements, not overlays"
- Ask for "monochrome with selective color application"
